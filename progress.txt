# MPL Build Progress

## Learnings
- Tailwind v4 uses CSS-based `@theme` for custom colors, not just `tailwind.config.ts`. Use `@config` directive in globals.css to link the config.
- `create-next-app` refuses project names with capitals — use temp dir and copy approach
- When copying from `/tmp`, symlinks in `node_modules/.bin/` break — must `rm -rf node_modules && npm install` after copy
- Vitest needs `passWithNoTests: true` in config to avoid exit code 1 when no test files exist yet
